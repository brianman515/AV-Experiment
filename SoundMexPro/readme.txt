Installation Instructions
-------------------------

- Run the installer SOUNDMEXPRO.EXE and follow the instructions
- Attach the dongle to the computer
- Add the BIN subdirectory of your installation to the MATLAB search path permanently.

NOTE: if you are using MATLAB 7.1 or below please rename the files

  BIN\soundmexpro.old.dll
  BIN\mpluginmex.old.dll

to
  BIN\soundmexpro.dll
  BIN\mpluginmex.dll
